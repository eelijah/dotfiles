# [Homebrew](https://brew.sh/index_ru.html)

## Instalation:

Instal Homebrew

```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" 
```

Search a new app: 
```sh
brew search ...
```
